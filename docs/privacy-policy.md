# MiniWeather 개인정보 처리방침

**최종 수정일: 2026년 4월 5일**

## 1. 개요

MiniWeather(이하 "앱")는 CHOI YEONSOON(이하 "개발자")이 운영하는 날씨 정보 제공 애플리케이션입니다. 본 개인정보 처리방침은 앱이 수집하는 정보와 그 사용 방법을 설명합니다.

## 2. 수집하는 정보

### 2.1 위치 정보
- **수집 항목**: 대략적인 위치 정보 (GPS 좌표)
- **수집 목적**: 사용자의 현재 위치에 해당하는 도시의 날씨 정보를 제공하기 위함
- **수집 방식**: 사용자가 위치 권한을 허용한 경우에만 수집
- **보관 기간**: 앱 내에서만 일시적으로 사용되며, 외부 서버에 저장하지 않음

### 2.2 날씨 데이터 캐시
- **수집 항목**: OpenWeatherMap API를 통해 조회한 날씨 데이터
- **수집 목적**: 오프라인 모드 지원 및 API 호출 최적화
- **보관 기간**: 30분간 로컬 기기에 캐시, 이후 자동 삭제

## 3. 수집하지 않는 정보

앱은 다음 정보를 수집하지 않습니다:
- 개인 식별 정보 (이름, 이메일, 전화번호 등)
- 사용자 계정 정보
- 결제 정보
- 연락처, 사진, 기타 기기 데이터
- 광고 식별자
- 사용 행태 분석 데이터

## 4. 제3자 서비스

앱은 날씨 데이터를 제공하기 위해 다음 제3자 서비스를 사용합니다:

- **OpenWeatherMap API**: 실시간 날씨 데이터를 조회하기 위해 사용자의 위치 좌표(위도, 경도)를 OpenWeatherMap 서버에 전송합니다. OpenWeatherMap의 개인정보 처리방침은 [여기](https://openweather.co.uk/privacy-policy)에서 확인할 수 있습니다.

## 5. 데이터 공유

앱은 사용자의 개인정보를 제3자에게 판매, 거래, 임대하지 않습니다. 날씨 데이터 조회를 위해 위치 좌표가 OpenWeatherMap API에 전송되는 것 외에 어떠한 데이터도 외부에 공유되지 않습니다.

## 6. 데이터 보안

모든 데이터는 사용자의 기기에 로컬로 저장되며, 외부 서버에 전송하거나 저장하지 않습니다. API 통신은 HTTPS를 통해 암호화됩니다.

## 7. 아동의 개인정보

앱은 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

## 8. 사용자의 권리

- 위치 권한은 기기 설정에서 언제든지 해제할 수 있습니다.
- 캐시 데이터는 앱 삭제 시 자동으로 제거됩니다.

## 9. 개인정보 처리방침의 변경

본 개인정보 처리방침은 변경될 수 있으며, 변경 시 앱 업데이트를 통해 고지합니다.

# MiniWeather Privacy Policy

**Last Updated: April 5, 2026**

## Overview

MiniWeather ("the App") is a weather application developed by CHOI YEONSOON ("the Developer"). This privacy policy explains what information the App collects and how it is used.

## Information We Collect

### Location Data
- The App collects approximate location data (GPS coordinates) solely to provide weather information for the user's current city.
- Location data is collected only when the user grants location permission.
- Location data is used temporarily within the App and is not stored on any external server.

### Weather Data Cache
- Weather data retrieved from OpenWeatherMap API is cached locally on the device for up to 30 minutes to support offline mode.

## Information We Do Not Collect

The App does not collect: personal identification information, account credentials, payment information, contacts, photos, advertising identifiers, or analytics data.

## Third-Party Services

The App uses OpenWeatherMap API to retrieve weather data. Location coordinates (latitude, longitude) are sent to OpenWeatherMap servers for this purpose. See [OpenWeatherMap Privacy Policy](https://openweather.co.uk/privacy-policy).

## Data Sharing

We do not sell, trade, or rent users' personal information to third parties.
